# Краткое руководство по Git
## 1. Основные команды
* git init - создаёт репозиторий
* git status - выводит информацию о состоянии репозитория
## 2. Работа с ветками
* git branch - выводит список веток
* git branch branch_name - создаёт ветку с именем branch_name
